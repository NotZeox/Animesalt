Retrieves complete homepage data including spotlights, trending anime, latest updates, top rated, ongoing series, movies, series, and recent episodes. Now with dynamic section discovery and non-empty array guarantee.

GET /api/info?id={id} Detailed anime information with metadata ‚ñº

Retrieves detailed information about a specific anime including synopsis, genres, languages, studio, seasons, and episodes with sub/dub flags.

Parameter Type Required Description
id string Yes The anime ID (slug format)
Response Example
{ "success": true, "data": { "id": "anime-id", "title": "Anime Title", "contentType": "series", "animeInfo": { "Overview": "Synopsis...", "Studios": "Studio Name", "Status": "Completed" }, "languages": [ { "name": "Japanese", "hasSub": true, "hasDub": false }, { "name": "English", "hasSub": true, "hasDub": true } ], "studio": { "name": "Studio Name", "link": "https://..." }, "episodes": [ { "episode_no": 1, "isSubbed": true, "isDubbed": false, "isSpecial": false } ] } }
GET /api/stream?id={id}&episode={ep} Streaming links with multiple servers ‚ñº

Retrieves streaming links and video sources for a specific episode.

Parameter Type Required Description
id string Yes The anime ID
episode string Yes Episode number (e.g., "1" or "1x1")
Example Request
curl "https://your-api.com/api/stream?id=naruto-shippuden&episode=1x1"
